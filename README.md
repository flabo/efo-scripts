# EFO-scripts

## 掲載しているスクリプト

* ラベル表示
* 半角変換
* パスワードチェッカー
* tem">リアルタイム・アラート
* パスワードマスク解除
* クレジットカードブランド表示
* 離脱ブロック
* GPSを利用したEFO

## 前提とする環境

* node.js
* npm

## 初期設定

* コマンドラインから下記を実行してください。

```
npm -g install grunt bower
npm install
bower install
```

* サイト立ち上げ・表示

下記を実行してください。

```
grunt serve
```

[デモ一覧](http://localhost:9000/demo/)より、各EFOスクリプトのデモが確認できます。
